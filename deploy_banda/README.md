# App Banda Sinfónica Juvenil 🎺

Aplicación web progresiva diseñada para la gestión y promoción de la Banda Sinfónica Juvenil.

## ✨ Características
- **Navegación Premium**: Diseño moderno "Floating Pill" con Glassmorphism.
- **Mecenas**: Sección integrada para donaciones vía Vaki, Nequi y Daviplata.
- **Gestión Real-time**: Sincronización automática con Firebase Firestore.
- **Herramientas**: Metrónomo, afinador y círculo de quintas para estudiantes.

## 🚀 Despliegue
Este proyecto está configurado para **Continuous Deployment** (CD) a través de Netlify. Cualquier cambio en la rama principal se publica automáticamente.
